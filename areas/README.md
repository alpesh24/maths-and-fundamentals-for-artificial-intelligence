# Areas

This directory contains the subject map of the repository. The structure follows the taxonomy proposed in the research report and groups related subjects into five top-level domains.

## Domains

- [Core Math](core-math/README.md)
- [Core CS](core-cs/README.md)
- [Core Optimization and Signals](core-optimization-signals/README.md)
- [Core ML](core-ml/README.md)
- [Vision Stack](vision-stack/README.md)

## Usage

- Keep high-level navigation in the domain `README.md` files.
- Keep subject-specific context, prerequisites, and study notes in each area folder.
- Store local PDFs in the mirrored `books/` tree outside git tracking.
