# Core Math

Core Math contains the analysis and algebra foundations that support modern machine learning, optimization, and vision.

## Areas

- [Linear Algebra](linear-algebra/README.md)
- [Probability and Statistics](probability-and-statistics/README.md)
- [Real Analysis](real-analysis/README.md)
- [Functional Analysis](functional-analysis/README.md)
- [Graph Theory](graph-theory/README.md)

## Why This Domain Matters

These subjects provide the language of vectors, uncertainty, convergence, and function spaces. They are the mathematical base for machine learning, deep learning, optimization, and signal-based methods.
