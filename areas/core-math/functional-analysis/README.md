# Functional Analysis

Functional Analysis extends linear algebra and analysis into infinite-dimensional settings. It is useful for understanding operators, spaces of functions, and deeper mathematical views of learning and inverse problems.

## Position In The Map

- Domain: [Core Math](../README.md)
- Typical prerequisites: [Real Analysis](../real-analysis/README.md), [Linear Algebra](../linear-algebra/README.md)
- Supports: [Deep Learning](../../core-ml/deep-learning/README.md)
- Prerequisite for: [Deep Learning](../../core-ml/deep-learning/README.md)
