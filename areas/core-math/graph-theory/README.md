# Graph Theory

Graph Theory provides the language of relational structure, networks, and combinatorial connectivity. It plays a direct role in PGM, structured learning, and parts of computer vision.

## Position In The Map

- Domain: [Core Math](../README.md)

- Supports: [PGM](../../core-ml/pgm/README.md)
